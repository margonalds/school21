## 🛠️ Стек технологий
* **Языки:** C, C++, SQL, Bash
* **Инструменты & DevOps:** Docker, Git, CI/CD, Linux (Ubuntu/Alpine)
* **Анализ & Тестирование:** UML, BPMN, QA методологии (Тест-кейсы, Баг-репорты)

## 📌 Особенности обучения
* **Самообучение:** Отсутствие менторов и лекций — поиск информации и решений ведется самостоятельно.
* **Peer-to-Peer:** Каждый проект проходит жесткую проверку другими участниками платформы (peer-review) и автоматизированными тестами (Verter).
