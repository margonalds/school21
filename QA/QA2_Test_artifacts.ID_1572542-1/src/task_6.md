# Task 6 - Составить тестовый сценарий

## Тестовый сценарий: Поступление в «Школу 21»

### Предусловия:
	Пользователь не зарегистрирован
    Пользователь  не учился в школе 21 ранее
	Сайт https://21-school.ru/ доступен

### Входные данные:
	ФИО
    Дата рождения
    Пол
	Email
	Номер телефона
    Личные документы

### Шаги:
	1.	Пользователь открывает сайт «Школы 21»
	2.	Переходит в раздел «Как поступить»
	3.	Ознакамливается с информацией о процессе поступления
	4.	Нажимает кнопку «Поступить в школу»
	5.	Выбирает кампус, выбирает программу, вводить e-mail, нажимает кнопку, нажимает "Поступить".
	6.  Вводит придуманный им пароль.
	7.	Заполняет анкету личными данными (ФИО, пол, дата рождения, номер телефона), соглашается с условиями участия в конкурсе, нажимает далее.
    8.  Нажимает начать, проходит 10 уровней тестовой игры, нажимает "Далее".
    9.  Смотрит видео про "Школу 21".
    10. Загружает документы, нажимает "Сохранить".

### Ожидаемый результат:
	Заявка успешно отправлена
	Пользователь получает подтверждение

### Постусловия:
	Пользователь зарегистрирован
	Заявка сохранена в системе
